# Shellular Server

WebSocket relay server for [Shellular](https://shellular.dev) — relays messages between Shellular CLI host agent and the Shellular app client.

## Requirements

- Node.js 18+
- pnpm

## Development

### Install

```bash
pnpm install
```

### Run (watch mode)

```bash
pnpm run dev
```

Server listens on `0.0.0.0:3000` by default.

### Run (one-shot)

```bash
pnpm start
```
