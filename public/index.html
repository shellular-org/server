Everything you need

Your dev environment,
untethered

Shellular connects any device to your full development stack over a secure WebSocket relay — no VPN, no port forwarding needed.

⚡
Full Terminal Access
xterm.js-powered PTY sessions with full color output, resize support, and shell history. Your real shell, on any screen.
🗂️
Filesystem Explorer
Browse, read, write, rename and delete files on the remote machine — all path-traversal protected and streamed over WebSocket.
📡
Zero Config Relay
Run npx shellular-cli start, scan the QR code, and you're in. No firewall rules, no tunnelling setup.
📱
Mobile-First UX
Built as an Acode plugin — lives right inside your mobile editor. Sidebar integration, touch gestures, and adaptive layouts.
🔐
Secure Sessions
Each session uses a short-lived 8-char token. The relay holds nothing — messages are forwarded and dropped immediately.
🤖
AI-Ready
Designed for AI-assisted coding workflows. Feed your remote codebase into any LLM without leaving your mobile editor.
Get up and running

Three steps to
code anywhere

No accounts. No cloud subscription. Just a local CLI and a browser.

1
Start the CLI on your machine
Run the CLI agent on your laptop, desktop, or cloud server. It starts the WebSocket relay and generates a session token.
# works on macOS, Linux, Windows
npx shellular-cli start
2
Connect from any device
Scan the QR code or paste the URL into Shellular on your mobile. Enter the 8-char session token — you're connected.
# or open directly in your browser
https://your-server:3000/<token>
3
Code, commit, ship
Your full terminal and filesystem are live. Read files, run builds, commit code, and get AI help — all from your phone.
# your entire dev stack, anywhere
git commit -m "shipped from mobile"
